Color CSS Data Type

Color Definitions

Definition Value Description
Named Color color: keyword; Using a keyword (such as red or blue). All existing keywords represent plain, solid colors without transparency.
RGB() color: rgb(); The rgb() functional notation expresses a color according to its red, green, and blue components.
HSL() color: hsl(); The hsl() functional notation expresses an sRGB color according to its hue, saturation, and lightness components.

Alpha Value

The value of an alpha-value is given as either a number or a percentage and can be used in rgb and hsl making their Syntax become rgba() and hsla().

If given as a number, the useful range is 0 (fully transparent) to 1.0 (fully opaque), with decimal values in between; that is, 0.5 indicates that half of the foreground color is used and half of the background color is used. Values outside the range of 0 to 1 are permitted, but are clamped to lie within the range 0 to 1.

If the alpha value is given as a percentage, 0% corresponds to fully transparent while 100% indicates fully opaque.
